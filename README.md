# Algorytmy-Sortowania---Analiza-Wydajnosci

Funkcjonalności

- Implementacja wszystkich czterech algorytmów sortowania
- Pomiar czasu wykonania metodą `System.nanoTime()`
- Testy skalowalności na różnych rozmiarach tablic
- Wczytywanie danych testowych z konsoli lub pliku tekstowego
- Uśrednianie wyników z wielu prób na losowo wygenerowanych tablicach
- Analiza przypadków średnich i pesymistycznych dla każdego algorytmu

# Wyniki

Program generuje porównanie czasów wykonania dla różnych rozmiarów danych wejściowych, pozwalając na analizę złożoności czasowej każdego algorytmu w praktyce.
